Overview

Welcome to the living style guide for Chef Compliance. To get started please attach the below file to your working code file.

styles/base.scss

We plan to use the light-weight grid system from bourbon. Please view Bourbon Grid System for information regarding the grid system.

Typography

Our type faces include montserrat-light, and HelveticaNeue-Light with a fall back to arial and helvetica.

H1 is used for the heading of a parent page

H2 is used for the heading of a child page

H3 is used for table headings and labels

Use the SMALL tag for a teeny tiny label

P is used for all paragraph text

Color variations
